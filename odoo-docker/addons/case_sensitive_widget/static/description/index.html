This Widget will make string become to UPPERCASE, lowercase or ProperCase

Try it, love it or remove it.

List features:

1. Uppercase widget
Ex: <field name="name" widget="upper_case"/>


2. Lowercase widget
Ex: <field name="name" widget="lower_case"/>


3. ProperCase widget: It is uppercase the first letter of each word.
Ex: <field name="name" widget="proper_case"/>
